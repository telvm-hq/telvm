## Gluten-free (celiac) control — venue and caterer agreement

These items should be **written into the BEO or catering addendum**.

1. **No wheat flour** in the prep area for the service window (or dedicated GF line with documented separation).
2. **Dedicated cutting boards and color-coded knives** for GF service; no shared toaster with gluten bread.
3. **Fryer policy**: if anything containing gluten was ever fried in oil, **do not use that fryer** for GF items; safest is **no fryer** for this series.
4. **Thickeners**: no wheat roux; verify every **stock, sauce, spice blend, soy sauce substitute, mustard, vinegar** (no malt vinegar).
5. **Garnishes**: no fried onions from unknown packs; verify **blue cheese** sources if used.
6. **Ingredient retention**: keep **vendor labels** for one service cycle beyond the event date.
7. **Staff briefing**: 15-minute pre-shift with **written one-pager** on hidden gluten and cross-contact.
8. **Guest communication**: menu cards list major allergens; host script avoids **medical promises** (see legal copy pack).
