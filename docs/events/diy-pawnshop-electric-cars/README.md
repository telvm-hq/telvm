# DIY pawn-shop electric cars — five-session build series

Operational pack for **five hands-on sessions** that go from the simplest bench “proof of motion” build (scavenged parts) to a **radio-controlled vehicle on a longboard deck** (trucks + wheels as the final chassis). Maintain these working documents here; do not treat the original plan file in `.cursor/plans/` as the live source of truth for session content.

- [schedule.md](schedule.md) — calendar-style run order; one-line build target per session.
- [curriculum.md](curriculum.md) — per-session objectives, build steps, stretch, failure modes, homework, parts.
- [safety-and-batteries.md](safety-and-batteries.md) — PPE, heat/shorts, chemistries, fire bag, where not to improvise.
- [pawnshop-procurement.md](pawnshop-procurement.md) — scavenge map, avoid-list, standardization (connectors, bolt-on mounts).
- [liability-and-disclaimer.md](liability-and-disclaimer.md) — hobby/educational framing; private property / closed course; no warranties.

**Cadence**: set **Session 1** when you have a venue, tools, and a sober battery policy; weekly or biweekly works well. See [schedule.md](schedule.md).
