# GitHub social preview (maintainers)

GitHub shows a **repository image** when the repo link is shared (Slack, X, Discord, etc.).

1. Prepare or crop an image to **1280×640** (aspect ratio **1.91∶1**), PNG or JPG.
2. In the repo on GitHub: **Settings → General → Social preview → Edit**.
3. Upload the image and save.

You can reuse the mascot from [`telvm-mascot.png`](telvm-mascot.png) with padding/crop, or a dedicated banner. This file is documentation only; the upload step is done in the GitHub UI.
