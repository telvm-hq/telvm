# v0.1.0

Initial public OSS snapshot of the **telvm** companion: Phoenix control plane for Docker-backed agent workflows, **ProxyPlug** preview routing, **Machine API** (`/telvm/api`), **ExplorerLive**, LiveView **Machines** UI, Docker HTTP adapter + mocks, and ExUnit coverage via **`docker compose --profile test`**.

**API reference:** [`docs/agent-api.md`](../agent-api.md) (endpoints, SSE events, scope).

**Includes:** Apache-2.0, CI workflow, GHCR publish workflow for the optional **go-http-lab** image, community docs (contributing, security, code of conduct), Dependabot, and issue/PR templates.

**Run tests:** `docker compose --profile test run --rm companion_test`

Use this body when creating the GitHub **Release** for tag `v0.1.0` (edit as needed).
